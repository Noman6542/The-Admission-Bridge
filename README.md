# University Admission Platform

A modern web application to explore universities abroad and submit applications based on eligibility criteria like GPA and IELTS. Built with **Next.js**, **React**, **TailwindCSS**, and **Node.js/Express** with **MySQL** as the database.

## Features
- Browse universities with filters:
  - Country
  - Degree Level
  - Tuition Fee
  - Search by name
- Real-time eligibility check based on GPA and IELTS
- Submit applications securely
- Prevent duplicate applications per university
- Responsive UI with light/dark theme support
- Modern animations using Framer Motion
- Toast notifications for success/error messages

---

## Tech Stack

**Frontend**
- Next.js (React Framework)
- TailwindCSS (Styling)
- React Toastify (Notifications)
- Framer Motion (Animations)

**Backend**
- Node.js & Express.js
- MySQL (Database)
- CORS & JSON parsing middlewares
